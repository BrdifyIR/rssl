## 🛠️ اسکریپت مدیریت و دریافت گواهی RSSL

اسکریپت **RSSL** یک ابزار تعاملی و هوشمند تحت لینوکس (Bash) است که به شما امکان می‌دهد بدون درگیر شدن با دستورات پیچیده، برای انواع پنل‌های فیلترشکن و مسیرهای دلخواه خود گواهی SSL رایگان و معتبر دریافت کنید.

### ✨ قابلیت‌های کلیدی:
* **کاملاً تعاملی (منوی هوشمند):** بدون نیاز به وارد کردن آرگومان در ترمینال، مراحل را گام‌به‌گام (انتخاب پنل، ایمیل و دامنه) جلو ببرید.
* **پشتیبانی از پروتکل‌های چندگانه:** تلاش خودکار برای دریافت SSL از طریق `acme.sh` و سوئیچ هوشمند روی `certbot` در صورت بروز خطا.
* **شناسایی خودکار مسیر پنل‌ها:** پشتیبانی پیش‌فرض از مسیرهای استاندارد پنل‌های محبوب مانند **مرزبان (Marzban)، پاسارگاد (Pasarguard)، مرزنشین، X-UI، 3X-UI، Hiddify و غیره**.
* **پشتیبانی از چند دامنه:** امکان دریافت یک گواهی SSL همزمان برای چندین دامنه یا ساب‌دامنه (تنها با جدا کردن آن‌ها توسط فاصله).

## 🚀 دستور نصب سریع (One-Line Installer)

کافیست دستور زیر را به صورت کامل کپی کرده و در ترمینال سرور خود (با دسترسی root) پیست کنید تا اسکریپت به صورت خودکار دانلود، فعال و اجرا شود:

```bash
curl -sL [https://raw.githubusercontent.com/BrdifyIR/rssl/main/rssl.sh](https://raw.githubusercontent.com/BrdifyIR/rssl/main/rssl.sh) -o /usr/local/bin/rssl && chmod +x /usr/local/bin/rssl && rssl
