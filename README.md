# 🛡️ RSSL - ابزار هوشمند صدور و مدیریت گواهی SSL و ربات تلگرام

اسکریپت **RSSL** یک ابزار قدرتمند، خودکار و تعاملی برای سیستم‌عامل‌های لینوکس است که به شما امکان می‌دهد به راحتی برای دامنه‌های خود گواهی SSL رایگان (از طریق Let's Encrypt و ZeroSSL) صادر و مدیریت کنید. این ابزار به طور ویژه برای پنل‌های محبوب میزبانی و پروکسی بهینه‌سازی شده و دارای یک سیستم هوشمند اطلاع‌رسانی و مدیریت از طریق ربات تلگرام است.

---

## ✨ قابلیت‌های کلیدی (Features)

* **⚡ صدور دوگانه و هوشمند:** تلاش اول برای صدور گواهی از طریق `acme.sh` (روش Standalone) و سوییچ خودکار به `Certbot` در صورت بروز خطا.
* **🤖 مدیریت و کنترل با ربات تلگرام:**
    * ارسال وضعیت موفقیت‌آمیز بودن یا خطای صدور گواهی به تلگرام.
    * دارای دکمه شیشه‌ای (Inline Keyboard) اختصاصی برای تایید و تمدید آنی گواهی‌ها.
    * اجرای سرویس پس‌زمینه (Daemon) برای گوش دادن به دستورات تلگرام به صورت ۲۴ ساعته.
* **📂 سازگاری با پنل‌های محبوب:** شناسایی خودکار و کپی کردن فایل‌های گواهی (`privkey.pem` و `fullchain.pem`) در مسیرهای استاندارد پنل‌های:
    * Marzban (مرزبان)
    * Marzneshin (مرزنشین)
    * Pasarguard (پاسارگارد)
    * X-UI / 3X-UI / S-UI / Hiddify
    * OVPanel
    * Rebecca
    * امکان تعریف مسیر سفارشی (Custom Path)
* **⏰ تمدید خودکار (Cron Job):** تنظیم خودکار کرون‌جاب سیستم برای بررسی و اعلان وضعیت گواهی‌ها در اول هر ماه میلادی.
* **🖥️ منوی تعاملی و CLI:** امکان استفاده هم به صورت منوی گرافیکی/متنی (Interactive) و هم به صورت دستورات یک‌خطی در ترمینال.
* **🔧 بررسی پیش‌نیازها:** نصب خودکار وابستگی‌های سیستم از جمله `curl` ،`socat` ،`jq` و مدیریت پکیج‌ها (APT/DNF/YUM).
